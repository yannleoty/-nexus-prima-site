NEXUS PRIMA — 14-FILE GITHUB-READY PACKAGE

1. Extract this ZIP in the iPhone Files app.
2. In GitHub, open the repository and choose Add file > Upload files.
3. Select ALL 14 extracted files together.
4. Upload them directly to the repository ROOT, not inside another folder.
5. Replace files with the same names and commit the changes.

IMPORTANT:
- Upload the extracted files, not the ZIP itself.
- index.html, styles.css, script.js, mars.webp and space.webp must be at the root.
- CNAME must remain at the root for nexusprima.org.
- The public contact address is contact@nexusprima.org.
